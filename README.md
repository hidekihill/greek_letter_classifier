# greek_letter_classifier
Final project for DATASCI 3ML3 at McMaster University. Trained elementary fully connected and convolutional networks on
<a href=https://www.kaggle.com/datasets/katianakontolati/classification-of-handwritten-greek-letters/data>Kontolati's Greek letter dataset</a>. Identified common misclassifications and compared model performance.
